 ------------------------
 Program Description
 ------------------------
 The following program aims to solve the NQueen Problem using the minimal conflict algorithm. It will generate a random 25
 Problem and display it to the user on execution followed by the Solved Board and it's fitness as well as the amount of time
 required to solve it following the minimal conflict Algorithm.


 -------------------------
 How To Run
 -------------------------
 1. Open Command Prompt from the Start Menu > CMD
 2. Type in the following commands:
	cd <filepath of folder where the program resides> 
	javac Main.java NQueenMinConflict.java QueenBoard.java
	java Main

 3. The program file contains a copy of precompiled class folders so it is not necessary to call javac
